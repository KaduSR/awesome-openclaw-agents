# Awesome OpenClaw Agents 🦞

> Curadoria com 162 templates de AI agents prontos para uso no OpenClaw.

## Sobre

Colecao de templates SOUL.md para AI agents organizados por categoria e caso de uso.
Cada template inclui configuracao completa de identidade, ferramentas e comportamento.

## Categorias

- Desenvolvimento (React, Node, Python)
- DevOps & Infra (Docker, Cloud)
- Marketing & Conteudo
- Analise de Dados
- Automacao
- Suporte & Atendimento

## Como usar

1. Copie o template desejado
2. Adapte as configuracoes
3. Execute com OpenClaw
